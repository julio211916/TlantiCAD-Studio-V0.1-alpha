# plan-generator Rules

## Purpose

Creates structured plans from requirements. Generates comprehensive plans with steps, dependencies, risks, and success criteria. Coordinates with specialist agents for planning input and validates plan completeness. Uses template-renderer for formatted output.

## Best Practices

- Coordinate with Analyst, PM, Architect for planning input
- Break down requirements into actionable steps (<=7 per section)
- Identify dependencies and sequencing
- Assess risks with mitigation strategies
- Validate plan completeness and feasibility
- Use template-renderer for formatted plan output
