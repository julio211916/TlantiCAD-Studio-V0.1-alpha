# Figma Research Requirements (2026)

## Verified API Endpoints

- GET /files/FILE_KEY — Get full file
- GET /files/FILE_KEY/nodes — Specific nodes
- GET /images/FILE_KEY — Export images
- GET /files/FILE_KEY/styles — Published styles
- GET /files/FILE_KEY/components — Components list
