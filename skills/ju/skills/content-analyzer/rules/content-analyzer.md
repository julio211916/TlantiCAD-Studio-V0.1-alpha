# content-analyzer Rules

## Purpose

Multi-dimensional content analysis: sentiment, readability, structure, topics, wording, engagement.

## Best Practices

- Always analyze all six dimensions
- Store results in content-analytics.json
- Compare against 7-day and 30-day baselines
- Classify hook type before analyzing engagement correlation

## Analysis Dimensions

1. Sentiment: Polarity, emotion, intensity
2. Readability: Flesch-Kincaid, sentence length, vocabulary
3. Structure: Hook, headings, visual breaks, CTAs
4. Topics: Primary, secondary, keywords, cluster
5. Wording: Power words, transitions, jargon
6. Engagement: Metrics, correlations, confidence
