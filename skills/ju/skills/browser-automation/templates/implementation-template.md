# browser-automation Implementation Template

## Goal

- Navigate to URLs and extract data
- Fill forms and submit them
- Capture screenshots and PDFs
- Manage authenticated sessions

## TDD

1. Red - Write tests for navigation, extraction, form filling
2. Green - Implement Playwright automation scripts
3. Refactor - Add anti-detection, error handling, retry logic

## Verification

- lint
- format
- targeted tests
- Integration tests against test websites

1. Set up Playwright Python environment
2. Implement navigation and waiting patterns
3. Implement data extraction with selectors
4. Implement form filling with label-based locators
5. Implement screenshot and PDF capture
6. Implement auth state persistence
7. Add anti-detection measures
