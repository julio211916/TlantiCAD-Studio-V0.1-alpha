# schema-creator Rules

## Purpose

Creates JSON Schema validation files for skills, agents, hooks, workflows, and data structures. Ensures type safety and input validation across the framework.

## Best Practices

- Use JSON Schema draft-07 or later
- Include descriptions for all properties
- Define required vs optional clearly
- Add examples in schema
- Use $ref for reusable definitions
- Validate schemas before saving
