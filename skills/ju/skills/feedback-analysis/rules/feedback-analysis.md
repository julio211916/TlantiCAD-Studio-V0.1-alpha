# feedback-analysis Rules

## Purpose

Analyze user feedback to extract actionable insights, identify patterns, and prioritize improvements.

## Best Practices

- Categorize feedback by type (bug, feature, usability, performance)
- Track sentiment trends over time
- Identify recurring themes and patterns
- Quantify impact (frequency, severity, user segments)
- Map feedback to actionable tasks
